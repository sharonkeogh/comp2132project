Game Play

  • Click on Roll Dice button to start round.
  • Score from 3 rolls is totalled to determine winner.
  • Click on New Game to play again.

Scoring

  • Dice values are added together for each round
  • However, if a 1 is rolled, score is 0 for round.
  • Also, if a pair of same numbers is rolled, total is doubled.
